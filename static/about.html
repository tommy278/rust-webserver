About

  • Multi-threaded Uses a thread pool (4 workers) to handle multiple requests simultaneously without blocking.
  • Manual HTTP Parsing: This server manually parses HTTP request strings to determine routes and methods.
  • MPSC Channel Architecture: Uses std::sync::mpsc to safely pass TCP streams from the main listener to worker threads.
  • Custom Routing: Logic handles .html, .css, and .js files with appropriate type headers.
